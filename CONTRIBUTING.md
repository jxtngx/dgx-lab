# Contributing

DGX Lab is a personal project. It is not a product, not maintained for general consumption, and not open to contributions.

You are welcome to fork or clone the repo, but if you do:

- Issues, bugs, and feature requests will not be triaged or addressed.
- Pull requests will not be reviewed or merged.
- You are on your own for setup, configuration, and troubleshooting.

This project is built for a specific hardware setup (NVIDIA DGX Spark) and a small group of users. If it is useful to you as-is, great. If not, fork it and make it yours.
